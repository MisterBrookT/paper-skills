# Venue Packs

Journal- and discipline-specific packs.

Possible packs:

- Nature-family;
- Cell-family;
- IEEE/ACM;
- economics and finance;
- circuits;
- biology;
- materials.

Venue packs should define field-specific writing, figure, citation, and review
expectations without duplicating shared workflow logic.

## Scope

- journal or conference positioning;
- expected contribution style;
- figure and table conventions;
- citation density and canonical references;
- common reviewer objections and rebuttal patterns.

## Demo Prompt

```text
Prepare a venue-fit checklist for this manuscript targeting an IEEE circuits
journal. Focus on contribution framing, figure expectations, experiment
evidence, citation norms, and likely reviewer objections.
```

## Integration Bias

Venue packs should compose `writing`, `citation`, `review`, and `plot` rules.
They should not fork shared workflow logic unless a venue demands it.
