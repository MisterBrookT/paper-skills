# Review

Reviewer simulation, rebuttal drafting, manuscript QA, and revision-response
workflows.

Default strategy: integrate or reference strong public review and response
skills before rebuilding local versions.

## Scope

- simulate reviewers by discipline and venue;
- turn reviewer comments into action plans;
- draft rebuttal letters with evidence-linked responses;
- run pre-submission QA for claims, figures, limitations, and contribution.

## Demo Prompt

```text
Act as a critical reviewer for an AI systems paper. Identify major weaknesses,
minor issues, missing experiments, and unclear claims. Return comments in the
style of a conference review, then propose a revision plan.
```

## Integration Bias

Use external review workflows where possible. Local value should come from
discipline-aware rubrics and response templates.
