# Writing

Curated writing, polishing, translation, and manuscript argumentation skills.

Default strategy: integrate or reference strong public academic writing skills
before rebuilding local versions.

## Scope

- section-level polishing without changing scientific claims;
- Chinese-English academic translation with field vocabulary preserved;
- argument repair for introduction, method, result, and discussion sections;
- clarity passes for abstracts, cover letters, and response letters.

## Demo Prompt

```text
Polish this introduction for a journal paper in materials science. Preserve all
technical claims, keep citations as placeholders, and return: revised text,
claim-risk notes, and a short rationale for major edits.
```

## Integration Bias

Use upstream writing skills through adapters first. Local rules should focus on
discipline vocabulary, claim preservation, and journal-specific tone.
